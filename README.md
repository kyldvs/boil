Boiler Plate Application
========================

This is some boiler plate for creating web applications.

Libraries Used
--------------

+ Node
+ Express
+ Coffeescript
+ Grunt
  + Coffeelint
  + Contrib-coffee
  + Contrib-copy
  + Contrib-clean
+ Jquery
+ Require
+ Backbone

Setup
-----

1. install npm packages
  npm install
2. build application
  grunt
3. start application
  npm start
