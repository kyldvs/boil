Boiler Plate Application
========================

This is some boiler plate for creating web applications.

### 1. Libraries Used

  + Node
  + Express
  + Jade
  + Coffeescript
  + Grunt
  + Jquery
  + Backbone
  + Stylus
  + Underscore
  + React
  + JSX

### 2. Setup

  1. install npm packages: `npm install`
  2. build application: `grunt`
  3. start application: `npm start`
